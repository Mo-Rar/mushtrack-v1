# Checklist Beta MushTrack

## A tester avant partage

- Ouvrir l'application sur mobile.
- Changer mode Hiver / Ete.
- Selectionner la team active dans l'onglet Chiens.
- Ajouter un chien.
- Demarrer une sortie GPS.
- Terminer la sortie.
- Verifier que le bilan post-entrainement s'ouvre.
- Autoriser la localisation pour tester la meteo auto.
- Enregistrer une sortie.
- Verifier que les km se reportent sur les chiens.
- Ajouter une course dans Agenda.
- Verifier Analyse, Course et Conseils.

## Retours beta a collecter

- Telephone utilise.
- Navigateur utilise.
- Ce qui est clair.
- Ce qui est confus.
- Donnees manquantes sur les chiens.
- Donnees manquantes sur les courses.
- Bugs ou lenteurs.

## Priorites apres beta

- Vrai GPS avec trace carte.
- Export GPX.
- Import/export des donnees.
- Sauvegarde cloud.
- Comptes utilisateurs.
- Version Android Play Store.
